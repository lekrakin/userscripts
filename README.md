# Userscripts for various sites and games

This repository contains userscripts I use on Tampermonkey for various sites and games.


## Clickpocalypse2 Auto

For [Clickpocalypse2](http://minmaxia.com/c2/). A personalized fork of [Clickpocalypse2Clicker](https://github.com/Clickpocalypse2Clicker/Clickpocalypse2Clicker).

## pkmn-idle++

For [Pokemon Idle](https://pkmn-idle.com/).
- Improves UI formatting.
- Adds a toggle button to minimize the PC Box.
